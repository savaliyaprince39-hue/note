Java Notes

Introduction

Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995. It is platform-independent, secure, and widely used for web, mobile, and enterprise applications.

Key Features

  • Simple and easy to learn
  • Object-Oriented
  • Platform Independent (Write Once, Run Anywhere)
  • Secure and Robust
  • Multithreaded
  • Rich Library Support

Basic Syntax

  • Java programs are made up of classes and methods.
  • The main method is the entry point of any Java application.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Object-Oriented Concepts

  • Class: Blueprint for objects.
  • Object: Instance of a class.
  • Inheritance: Deriving new classes from existing ones.
  • Polymorphism: Ability to take many forms.
  • Encapsulation: Wrapping data and methods together.
  • Abstraction: Showing essential features, hiding implementation details.
